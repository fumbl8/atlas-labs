# Phylogenetic Signal Notebook

**Toy sequence and trait data into introductory phylogenetic reasoning**

This learning notebook walks from alignment-like data to a distance matrix, a simple tree, trait mapping, and careful interpretation.

## What it demonstrates

- Phylogenetic workflow literacy.
- Distance-matrix and tree reasoning.
- Evolutionary-biology bridge between organismal and computational work.
- Explicit toy-to-real boundary setting.

## Public-facing boundary

This is a toy pedagogical workflow, not accepted taxonomy, model-based inference, or a validated phylogenetic analysis. It uses introductory methods and simplified trait coding.
