# BioCorridor

**Physarum-inspired habitat-network mapper**

BioCorridor is a small prototype that uses slime-mold-inspired adaptive trail formation to make habitat-connectivity hypotheses visible across fragmented landscapes.

## What it demonstrates

- Bio-inspired simulation.
- Agent-based network emergence.
- Habitat patches, landscape resistance, and corridor hypotheses.
- Static visual/report generation.
- Science communication with explicit validation limits.

## Public-facing boundary

BioCorridor is a portfolio prototype, not a conservation-planning tool. It does not claim species-specific movement validation, expert resistance surfaces, formal least-cost/circuit-theory comparison, management authority, or field review.
