# Source Note

This public repository is a cleaned presentation layer derived from a larger local-only Atlas Labs portfolio workspace.

The local workspace contains drafts, internal planning history, output folders, caches, and private operating notes. Those materials were intentionally not copied into this public repo.

What was preserved:

- the Atlas Labs direction,
- the WIP/early-portfolio posture,
- the project index,
- the biology/ecology/systems focus,
- the validation boundaries,
- and concise summaries of the strongest project lanes.

What was left out:

- private planning and handoff notes,
- local file paths,
- raw outputs and generated caches,
- scratch files,
- internal automation/system notes,
- imported-source metadata,
- and project files that need review before public release.
